
# 💖 For Krushnali - A Special Interactive Proposal Website

An elegant, interactive, and fully responsive single-page web application designed with love to ask a very special question. 

## 🚀 Live Demo
Check out the live website here: **[https://ravixtreme.github.io/For-Krushnali/]**

---

## ✨ Features
*   **The Uncatchable "No" Button:** An elusive "No" button that dynamically recalculates its position and flees whenever the cursor hovers over it or tries to click it. Saying "No" is mathematically impossible! 😉
*   **Celebration State:** Instant screen transition upon clicking "Yes!" featuring a celebratory dance animation.
*   **Floating Hearts Animation:** Smooth, dynamically generated CSS background animations that fill the screen with floating hearts upon acceptance.
*   **Direct Action Triggers:** One-click integration with mobile hardware to immediately trigger a phone call or open WhatsApp with a pre-filled sweet message.
*   **Fully Responsive:** Optimized for all screen sizes, from high-end mobile displays to desktop browsers.
